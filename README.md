# Instagram Reel Downloader

A simple web application that allows users to download Instagram Reels, videos, and posts by pasting the Instagram URL.

## 🚀 Features

* Download Instagram Reels
* Download Instagram Videos
* Simple and clean user interface
* Fast download process
* Works directly in the browser

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Library:** instagram-url-direct

## 📁 Project Structure

```
instagram-reels-downloader
│
├── server.js
├── package.json
└── public
    ├── index.html
    ├── style.css
    └── script.js
```

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/pradeepkrsurya01/instagram-reels-downloader.git
```

2. Go to project folder

```
cd instagram-reels-downloader
```

3. Install dependencies

```
npm install
```

4. Start the server

```
node server.js
```

5. Open in browser

```
http://localhost:3000
```

## 📌 How It Works

1. Copy the Instagram Reel or video link.
2. Paste the link into the input field.
3. Click the **Download** button.
4. The video will appear with a download option.
