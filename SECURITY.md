# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes     |
| < 1.0   | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

Please follow these steps:

1. **Do not open a public issue immediately.**
2. Contact the maintainer directly with details of the vulnerability.
3. Provide a clear description of the issue and steps to reproduce it.
4. If possible, include screenshots, logs, or proof of concept.

## Contact

For security concerns, please contact:

## Responsible Disclosure

We appreciate responsible disclosure of vulnerabilities and will work quickly to resolve any valid security issues.

Please allow reasonable time for fixes before publicly disclosing the issue.

## Scope

This policy applies to:

* The main application
* Backend API
* Frontend components
* Any dependencies included in this project
